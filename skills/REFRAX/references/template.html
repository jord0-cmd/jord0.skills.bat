REFRAX

"Stop reading code. Start seeing it."

REFRAX turns code into visual stories. Claude analyses your code, generates structured data, and this page renders it as an interactive logic spine with risk analysis and diff review.

Logic Spine

Click any node to inspect it. The right panel shows multiple views:

• Plain — What it does in everyday language
• Dev — Technical details for developers
• Code — Source code with highlighted lines
• Risks — All identified security and quality risks
• Diff — Code changes with explanations
• Glossary — Jargon-free term definitions

Navigating Risks

Click any risk card to jump to the affected node in the spine. Use Copy Fix Prompt to get a ready-made Claude prompt to fix the issue.

Modes

Toggle NOV / PRO to switch between novice and professional language.

Features

• Confidence Score — Hover the badge in the header to see how confident the analysis is
• Tech Stack — Detected technologies shown as chips; hover for evidence
• File Indicator — Shows which file a selected node belongs to
• Overview Button — Click « Overview to deselect and return to the summary view
• Live Updates — When the analysis is refined, the diagram morphs automatically — no refresh needed
• Glossary Auto-Linking — Underlined terms in text show definitions on hover

Step — An action or operation
Decision — A branch point
Side Effect — External interaction (DB, network, file)
Success — Happy path outcome
Failure — Error or rejection
