# Lighting Keywords Reference

## Natural Light

```
golden hour           - warm, long shadows, magic hour
blue hour             - cool twilight, purple-blue tones
harsh midday sun      - high contrast, sharp shadows
overcast              - soft, even, diffused
dappled sunlight      - through foliage, spotted patterns
backlit               - silhouette, rim glow
```

## Studio/Artificial Light

```
three-point lighting  - professional, balanced
Rembrandt lighting    - triangle highlight on cheek
split lighting        - half face illuminated
butterfly lighting    - glamorous, under-eye shadows
rim lighting          - edge definition, separation
soft box lighting     - even, flattering
spotlight             - theatrical, isolated
```

## Atmospheric Effects

```
volumetric light      - god rays, visible beams
atmospheric haze      - depth, distance
fog, mist             - mystery, ethereal
smoke                 - texture, drama
dust particles        - realism, texture
lens flare            - cinematic stylization
neon glow             - urban, cyberpunk
```

## Specific Lighting Descriptions (Advanced)

Instead of "dramatic lighting", try:
```
"single key light positioned 45 degrees above left of subject, creating natural shadow fall on right side of face, secondary fill light at 25% intensity"
```
