# Composition Keywords Reference

## Camera Angles

```
eye-level shot        - neutral, relatable
low-angle shot        - powerful, imposing subject
high-angle shot       - vulnerable, diminished subject
Dutch angle           - tension, unease
bird's-eye view       - environmental context, overhead
worm's-eye view       - extreme low, dramatic
over-the-shoulder     - POV perspective
```

## Shot Framing

```
extreme close-up (ECU)  - eyes only, intense emotion
close-up (CU)           - face fills frame
medium close-up (MCU)   - head and shoulders
medium shot (MS)        - waist up
medium long shot (MLS)  - knees up
long shot (LS)          - full body with environment
extreme long shot (ELS) - tiny figure in vast landscape
```

## Lens Specifications

```
24mm wide-angle       - exaggerated perspective, environmental
35mm                  - documentary feel, natural context
50mm                  - natural perspective, classic
85mm portrait lens    - flattering compression, soft bokeh
135mm telephoto       - compressed background, isolation
macro lens            - extreme detail, intimate scale
fisheye               - distorted, 180-degree view
```

## Composition Rules

```
rule of thirds, golden ratio, centered composition,
leading lines, frame within a frame, negative space,
symmetrical balance, foreground interest, depth layering
```
