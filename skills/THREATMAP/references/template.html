THREATMAP

Visual Threat Model Generator — Force-directed security architecture with STRIDE-categorized threats.

Interactions

  • Click a node to inspect its details, threats, and data flows
  • Click a threat barnacle to see severity, STRIDE category, and blast radius
  • Hover a node to highlight connected data flows
  • Drag nodes to reposition them in the force simulation
  • Esc to close the side panel and reset highlights
  • Controls button to deploy/retract security controls

STRIDE Categories

Spoofing — Identity impersonation
Tampering — Data modification
Repudiation — Action deniability
Information Disclosure — Data leakage
Denial of Service — Availability attacks
Elevation of Privilege — Unauthorized access

Severity Levels

Critical (9-10) — Vibrating marker
High (7-8.9) — Trembling marker
Medium (4-6.9) — Steady marker
Low (0.1-3.9) — Calm marker
Info — Dim marker

Node Types

Process — Hexagonal, pulsing core
Data Store — Rounded rect, layered
External Entity — Jagged, untrusted
Infrastructure — Shield/chevron

Attack Simulation

Click a threat barnacle, then "Simulate Attack" in the panel. Watch tendrils probe, exploit arcs fire, and lateral movement spread through the graph. Compromised nodes glow red with persistent scars.
