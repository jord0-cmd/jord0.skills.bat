REGTRAX

AI-Powered Regex Railroad Visualizer

How It Works

REGTRAX is an AI-partnered tool. You describe what you want to match in plain English in the terminal, and the AI builds the regex, explains it, and sends it here for visual testing.

You describe → AI writes regex → Diagram appears → You test

Testing Strings

  • Type a test string in the input box and press Enter
  • The photon traces the railroad diagram
  • Green = match — Red = derailment at the failure point
  • Use the scrubber below the diagram to step through manually

Live Updates

When you ask the AI to change the pattern, the diagram morphs automatically. No refresh needed — just keep talking to the AI and testing here.

Reading the Diagram

  • Cyan nodes — literal characters
  • Amber nodes — character classes like [a-z] or \d
  • Violet arcs — quantifiers (+ one or more, * zero or more, ? optional, {n} exact count)
  • Emerald borders — capture groups
  • Rose nodes — anchors (^ start, $ end, \b boundary)
  • Dashed cyan — backreferences (\1 matches same text as a capture group)
